# МГУПИ—90: цифровая капсула времени

Интерактивный сайт-капсула времени, разработанный для технологической практики.

Требуется Node.js 20.19 или новее.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Структура проекта

- `src/components` — компоненты интерфейса;
- `src/data` — данные временной линии и мини-квеста;
- `src/styles` — глобальная визуальная система;
- `public/images` — локальные изображения проекта.

## Функциональность

- интерактивная временная линия;
- мини-квест из трёх смысловых выборов с персональным результатом;
- интерактивный кроссворд с дополнительными намёками и поэтапным открытием букв;
- сохранение ответов и использованных подсказок кроссворда;
- форма послания в будущее;
- сохранение данных в `localStorage`;
- адаптивный интерфейс;
- полноэкранная галерея;
- печатная финальная карточка;
- локальные иконки Iconify без зависимости от внешней сети.

Проект работает без backend и не отправляет пользовательские данные на сервер.
